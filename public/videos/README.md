# Arkan video assets

Place looping MP4 files here for hero and page backgrounds. Do not commit large video files to git.

## Required files

| File | Usage |
|------|--------|
| `arkan-hero-market-entry-loop.mp4` | Homepage hero (8–12s loop, muted) |
| `arkan-services-loop.mp4` | Services page hero |
| `arkan-sectors-loop.mp4` | Sectors page hero |
| `arkan-insights-loop.mp4` | Insights page hero |
| `arkan-contact-loop.mp4` | Contact page hero |

Until files are added, pages fall back to poster images automatically.

## Production notes

- Keep loops 8–12 seconds, muted, no audio track required
- Export H.264 MP4, 1920×1080 or 1280×720
- Use real Libya operational footage where possible
- Dark navy overlay is applied in code — footage can be slightly bright
